Why this layout

  • 4-tile asset hero in navy/gold showing total net book value (GHS 1,004,600) with the down-from-year-start delta (depreciation expensed YTD = GHS 77,800). The hero teaches the concept: assets shrink each year through depreciation, and that shrinkage is a real expense.
  • Cross-reference card to GES annual census in gold-bordered tint: "One entry, two outputs." Recording assets here flips the 2-page hatched manual section in the annual census to auto-fill. This is the key value proposition for operators — work done once produces both a balance sheet AND fills statutory paperwork.
  • Category tabs with counts: All 22 / Buildings 3 / Vehicles 1 / Furniture 5 / ICT 8 / Kitchen 3 / Sanitation 2. 5 categories aligned with the GES annual census infrastructure section so the data model maps cleanly.
  • Asset table grouped by category with gold-bordered header rows showing category icon, name, useful-life convention, and subtotal. The convention "25-year straight line" is shown in the group header so admin understands why each building depreciates the way it does.
  • 6 columns: Asset / Acquired / Original cost / Depreciation / Condition / Book value. Acquisition date and cost in mono. Depreciation as a visual progress bar showing % depreciated + years remaining. Condition uses 3-state tag (Good green / Fair warn / Poor terra). Book value is the headline output number.
  • Depreciation bar uses three colour states: green for low (less than 33% depreciated), gold default, terra for high (over 75%). Visual ageing of assets — schools see at a glance which assets are nearing end-of-life and need replacement planning.
  • The Toyota Coaster school bus has full operational identity: "GR 4421-22 · 30-seater". Real Ghana number-plate format and capacity detail matters because the registry is also operational, not just accounting.
  • "ICT lab desktops · 9 working" condition note reflects the partial-functionality state from the annual census infrastructure section. Same vocabulary as the form: "Functional · all working / Partial · some working / Computers but no power / connectivity / No ICT lab" — captured here at asset level.
  • Fully-depreciated cooking pots (100% depreciated, still in use, GHS 0 book value) are an honest treatment of school operations. Pots bought in 2018 are still cooking lunch — the depreciation schedule says they're worth zero on paper, but they're operationally valuable. The register doesn't auto-retire them; admin marks disposal explicitly when an asset is sold or scrapped.
  • Sanitation / latrines tracked as the borehole + tank capital line item. KVIP latrine count appears in the group meta line. Sanitation as fixed asset rather than expense matches accounting reality and feeds the GES infrastructure section accurately.
  • Asset IDs follow CTK-CAT-NNN pattern (CTK-BLD-001, CTK-ICT-013, etc.) — visible in mono in the asset name cell. Schoolish asset numbering convention that operators can paint or sticker on the physical items for inventory tracking.
  • Total row at the bottom in italic Fraunces label "All categories · 22 assets" with GHS 1,004,600. The same number that appears in the hero — visual closure of the table.
